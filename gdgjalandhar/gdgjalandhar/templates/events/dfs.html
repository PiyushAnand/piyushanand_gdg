WHAT IS DART?

Dart is a new language created by Google to help you build modern web apps with core libraries and tools such as a package manager, editor, and a compiler to JavaScript. Dart has hit 1.0 and is now Flight Ready!

WHAT IS DART FLIGHT SCHOOL?

Dart Flight School is an outreach program, under the aegis of Google, taking place across many cities globally all through February. Here, you will learn Dart and get your web development wings.

HOW IS ANGULAR INVOLVED WITH THIS EVENT SERIES?

Angular has ported their popular framework to Dart, and has provided tutorials and examples.
